HTML DOM

HTML Documents

Os documentos HTML são arquivos que contêm a estrutura e o conteúdo de uma página da web. Eles podem ser acessados e manipulados usando JavaScript por meio do DOM.

Exemplo de código:

// Acessando propriedades do documento HTML
console.log(document.title); // Retorna o título do documento
console.log(document.URL); // Retorna a URL do documento
console.log(document.body); // Retorna o elemento body do documento

// Manipulando o conteúdo do documento HTML
document.title = "Novo Título"; // Define um novo título para o documento
document.body.style.backgroundColor = "red"; // Muda a cor de fundo do body para vermelho

// Acessando elementos do documento HTML
const header = document.getElementById("header"); // Obtém um elemento pelo seu ID
const paragraphs = document.getElementsByTagName("p"); // Obtém uma coleção de elementos por sua tag
const form = document.forms[0]; // Obtém o primeiro formulário do documento

// Manipulando elementos do documento HTML
header.textContent = "Novo Título"; // Modifica o texto do elemento header
paragraphs[0].style.color = "blue"; // Muda a cor do primeiro parágrafo para azul
form.submit(); // Submete o formulário

Propriedades:

document.title: Retorna ou define o título do documento.

document.body: Retorna o elemento <body> do documento.

document.head: Retorna o elemento <head> do documento.

document.documentElement: Retorna o elemento <html> do documento.

document.domain: Retorna o domínio do documento atual.

document.URL: Retorna a URL do documento atual.

document.referrer: Retorna a URL do documento que referenciou o documento atual.

document.characterSet: Retorna o conjunto de caracteres usado pelo documento.

document.images: Retorna uma coleção de elementos <img> do documento.

document.forms: Retorna uma coleção de elementos <form> do documento.

document.links: Retorna uma coleção de elementos <a> do documento.

document.scripts: Retorna uma coleção de elementos <script> do documento.

document.styleSheets: Retorna uma coleção de folhas de estilo do documento.

HTML Elements

Os elementos HTML são as tags que compõem a estrutura de um documento HTML. Cada tag representa um elemento diferente.Os elementos podem ser acessados e manipulados usando o DOM.

Exemplo de código:

// Acessando propriedades dos elementos HTML
const myElement = document.getElementById("myElement"); // Obtém um elemento pelo seu ID
console.log(myElement.id); // Retorna o ID do elemento
console.log(myElement.className); // Retorna a lista de classes do elemento
console.log(myElement.tagName); // Retorna a tag do elemento em letras maiúsculas

// Manipulando os elementos HTML
myElement.className = "highlight"; // Define uma classe para o elemento
myElement.innerHTML = "<b>Novo conteúdo</b>"; // Define um novo conteúdo HTML para o elemento

// Manipulando estilos dos elementos HTML
myElement.style.color = "red"; // Muda a cor do texto do elemento para vermelho
myElement.style.backgroundColor = "yellow"; // Muda a cor de fundo do elemento para amarelo

// Manipulando eventos dos elementos HTML
myElement.addEventListener("click", function() {
  console.log("O elemento foi clicado!");
});

// Criando novos elementos HTML
const newElement = document.createElement("div"); // Cria um novo elemento <div>
newElement.id = "newElement"; // Define o ID do novo elemento
newElement.textContent = "Novo Elemento"; // Define o texto do novo elemento

// Inserindo elementos no documento
const parentElement = document.getElementById("parentElement"); // Obtém o elemento pai
parentElement.appendChild(newElement); // Adiciona o novo elemento como filho do elemento pai

Propriedades:

  • attributes: Retorna uma coleção de atributos do elemento.
  • classList: Retorna uma lista de classes atribuídas ao elemento.
  • className: Retorna ou define a lista de classes do elemento.
  • clientHeight: Retorna a altura do elemento, incluindo o padding.
  • clientWidth: Retorna a largura do elemento, incluindo o padding.
  • id: Retorna ou define o valor do atributo id do elemento.
  • innerHTML: Retorna ou define o conteúdo HTML dentro do elemento, incluindo tags.
  • innerText: Retorna ou define o texto interno do elemento, excluindo tags HTML.
  • nodeName: Retorna o nome da tag do elemento em letras maiúsculas.
  • nodeType: Retorna o tipo do nó (geralmente 1 para elementos HTML).
  • offsetHeight: Retorna a altura total do elemento, incluindo o padding, borda e barra de rolagem.
  • offsetWidth: Retorna a largura total do elemento, incluindo o padding, borda e barra de rolagem.
  • offsetLeft: Retorna a posição horizontal do elemento em relação ao elemento pai deslocado.
  • offsetTop: Retorna a posição vertical do elemento em relação ao elemento pai deslocado.
  • parentNode: Retorna o nó pai do elemento.
  • style: Retorna um objeto que contém as propriedades CSS calculadas e/ou definidas do elemento.
  • tagName: Retorna o nome da tag do elemento em letras maiúsculas.
  • textContent: Retorna ou define o texto do conteúdo do elemento, incluindo texto e nós filhos.

HTML Attributes

Os atributos HTML são usados para fornecer informações adicionais sobre os elementos. Eles são definidos dentro das tags HTML e podem conter valores que afetam o comportamento ou a aparência dos elementos.

Exemplo de código:

// Acessando e manipulando atributos HTML
const link = document.getElementById("myLink");

console.log(link.getAttribute("href")); // Retorna o valor do atributo href
console.log(link.hasAttribute("target")); // Verifica se o elemento possui o atributo target

link.setAttribute("href", "https://www.example.com"); // Define um novo valor para o atributo href
link.setAttribute("target", "_blank"); // Define o atributo target com o valor _blank

link.removeAttribute("title"); // Remove o atributo title, se existir

// Verificando a presença de atributos HTML
const image = document.querySelector("img");

if (image.hasAttribute("alt")) {
  console.log("A imagem possui o atributo alt");
} else {
  console.log("A imagem não possui o atributo alt");
}

Propriedades:

  • getAttribute(name): Retorna o valor do atributo especificado pelo nome.
  • setAttribute(name, value): Define o valor do atributo especificado pelo nome.
  • hasAttribute(name): Verifica se o elemento possui o atributo especificado pelo nome.
  • removeAttribute(name): Remove o atributo especificado pelo nome.
  • getAttributeNames(): Retorna uma matriz contendo os nomes de todos os atributos do elemento.
  • getAttributeNode(name): Retorna o objeto de nó do atributo especificado pelo nome.
  • setAttributeNode(attrNode): Define o objeto de nó do atributo no elemento.
  • removeAttributeNode(attrNode): Remove o objeto de nó do atributo do elemento.
  • hasAttributes(): Verifica se o elemento possui algum atributo.
  • classList.add(className): Adiciona uma classe CSS ao elemento.
  • classList.remove(className): Remove uma classe CSS do elemento.
  • classList.toggle(className): Alterna a presença de uma classe CSS no elemento.
  • classList.contains(className): Verifica se o elemento possui uma classe CSS específica.
  • classList.item(index): Retorna a classe CSS no índice especificado.
  • classList.length: Retorna o número de classes CSS no elemento.

HTML Collection

HTML Collection é um objeto que representa uma coleção de elementos HTML, como os obtidos por métodos.Essa coleção é semelhante a uma matriz e pode ser acessada e iterada usando índices.

Exemplo de código:

// Obtendo elementos usando métodos que retornam uma HTML Collection
const paragraphs = document.getElementsByTagName("p"); // Obtém todos os elementos <p>
const buttons = document.getElementsByClassName("btn"); // Obtém todos os elementos com a classe "btn"

// Acessando propriedades da HTML Collection
console.log(paragraphs.length); // Retorna o número de parágrafos no documento

// Acessando elementos da HTML Collection
const firstParagraph = paragraphs[0]; // Obtém o primeiro parágrafo
const secondButton = buttons.item(1); // Obtém o segundo botão

// Manipulando elementos da HTML Collection
firstParagraph.style.color = "red"; // Muda a cor do primeiro parágrafo para vermelho
secondButton.disabled = true; // Desabilita o segundo botão

// Iterando sobre uma HTML Collection
for (let i = 0; i < paragraphs.length; i++) {
  const paragraph = paragraphs[i];
  console.log(paragraph.textContent); // Imprime o texto de cada parágrafo
}

Propriedades:

  • item(index): Retorna o elemento na posição especificada pelo índice.
  • namedItem(name): Retorna o elemento com o nome ou ID especificado.
  • length: Retorna o número de elementos na coleção.
  • [index]: Permite acessar um elemento da coleção diretamente pelo seu índice.
  • entries(): Retorna um iterador de matriz que contém pares [índice, elemento] para cada item na coleção.
  • forEach(callback): Executa uma função de retorno de chamada para cada elemento na coleção.
  • keys(): Retorna um iterador de matriz que contém os índices dos elementos na coleção.
  • values(): Retorna um iterador de matriz que contém os elementos na coleção.

HTML NodeList

NodeList é um objeto que representa uma lista de nós do DOM, como os obtidos por métodos.Assim como o HTML Collection, o NodeList é semelhante a uma matriz e pode ser acessado e iterado usando índices.

Exemplo de código:

// Selecionando elementos com querySelectorAll
const paragraphs = document.querySelectorAll("p"); // Retorna uma NodeList de todos os elementos <p>

// Acessando propriedades do NodeList
console.log(paragraphs.length); // Retorna o número de elementos <p> encontrados

// Iterando sobre a NodeList
for (let i = 0; i < paragraphs.length; i++) {
  console.log(paragraphs.item(i)); // Imprime cada elemento <p>
}

// Acessando elementos específicos do NodeList
const firstParagraph = paragraphs.item(0); // Obtém o primeiro elemento <p>
const lastParagraph = paragraphs.item(paragraphs.length - 1); // Obtém o último elemento <p>

Propriedades:

item(index): Retorna o nó na posição especificada pelo índice.

forEach(callback): Executa uma função de retorno de chamada para cada nó na NodeList.

keys(): Retorna um novo objeto Array Iterator que contém as chaves para cada elemento na NodeList.

values(): Retorna um novo objeto Array Iterator que contém os valores para cada elemento na NodeList.

entries(): Retorna um novo objeto Array Iterator que contém os pares chave/valor para cada elemento na NodeList.

forEach(callback, thisArg): Executa uma função de retorno de chamada para cada nó na NodeList, permitindo especificar o valor de this dentro da função.

[Symbol.iterator](): Retorna um novo objeto Array Iterator que contém os valores para cada elemento na NodeList.

Symbol.toStringTag: Propriedade que retorna uma string representando a tag de objeto padrão.

HTML DOMTokenList

DOMTokenList é um objeto que representa uma lista de classes CSS em um elemento HTML. Ele fornece métodos para adicionar, remover e manipular classes em um elemento.

Exemplo de código:

// Acessando propriedades do DOMTokenList
const element = document.getElementById("myElement");
const classList = element.classList;

console.log(classList.length); // Retorna o número de classes na lista
console.log(classList.value); // Retorna uma representação de string das classes

// Manipulando a lista de classes
classList.add("class1"); // Adiciona a classe "class1"
classList.add("class2", "class3"); // Adiciona as classes "class2" e "class3"
classList.remove("class2"); // Remove a classe "class2"
console.log(classList.contains("class1")); // Verifica se a classe "class1" está presente (true)
classList.toggle("class1"); // Remove a classe "class1" (já que estava presente)
classList.replace("class3", "class4"); // Substitui a classe "class3" por "class4"

// Iterando sobre a lista de classes
for (let i = 0; i < classList.length; i++) {
  const token = classList.item(i);
  console.log(token); // Imprime cada classe na lista
}

// Convertendo a lista de classes em uma string
const classString = classList.toString();
console.log(classString); // Retorna uma string com as classes separadas por espaço

Propriedades:

  • length: retorna o número de classes na lista.
  • value: uma representação de string das classes contidas na lista.

Métodos:

  • add(token1, token2, ...): adiciona um ou mais tokens (classes) à lista.
  • remove(token1, token2, ...): remove um ou mais tokens (classes) da lista.
  • contains(token): verifica se a lista contém um token (classe) específico.
  • toggle(token): adiciona um token à lista se ele não estiver presente e o remove se já estiver presente.
  • replace(oldToken, newToken): substitui um token antigo por um novo na lista.
  • item(index): retorna o token (classe) na posição especificada pelo índice.
  • toString(): retorna uma string que representa a lista de tokens.

HTML Styles

Os estilos HTML controlam a aparência dos elementos em uma página da web. Eles podem ser definidos inline usando o atributo style ou em um arquivo de estilo externo (CSS). Os estilos podem ser acessados e modificados usando o DOM.

Exemplo de código:

<!DOCTYPE html>
<html>
<head>
  <style>
    .my-element {
      color: blue;
      font-size: 18px;
    }
  </style>
</head>
<body>
  <div id="myDiv" class="my-element">Exemplo de Div</div>

  <script>
    const myElement = document.getElementById("myDiv");
    
    // Acessando propriedades de estilo do elemento
    console.log(myElement.style.color); // Retorna a cor definida no estilo inline ou vazia se não estiver definida
    console.log(myElement.style.fontSize); // Retorna o tamanho da fonte definido no estilo inline ou vazio se não estiver definido

    // Modificando propriedades de estilo do elemento
    myElement.style.color = "red"; // Modifica a cor do texto para vermelho
    myElement.style.fontSize = "20px"; // Modifica o tamanho da fonte para 20 pixels

    // Adicionando classes para alterar estilos
    myElement.classList.add("another-class"); // Adiciona uma classe adicional ao elemento
    myElement.classList.remove("my-element"); // Remove a classe original do elemento
  </script>
</body>
</html>

Propriedades:

  • cssText: retorna ou define todas as regras CSS em forma de texto para o elemento.
  • length: retorna o número de propriedades de estilo definidas para o elemento.
  • parentRule: retorna a regra CSS pai, se aplicável.

Métodos:

  • getPropertyPriority(propertyName): retorna a prioridade de uma propriedade CSS específica para o elemento.
  • getPropertyValue(propertyName): retorna o valor de uma propriedade CSS específica para o elemento.
  • removeProperty(propertyName): remove uma propriedade CSS específica do estilo do elemento.
  • setProperty(propertyName, value, priority): define ou modifica o valor de uma propriedade CSS específica para o elemento.

Web APIs

API Canvas

A API Canvas permite desenhar gráficos, animações e imagens usando JavaScript. Ela fornece métodos e propriedades para criar e manipular elementos canvas no DOM.

Exemplo de código:

// Obtendo o elemento canvas
const canvas = document.getElementById("myCanvas");

// Obtendo o contexto 2D do canvas
const ctx = canvas.getContext("2d");

// Definindo a largura e altura do canvas
canvas.width = 500;
canvas.height = 300;

// Desenhando um retângulo preenchido
ctx.fillStyle = "red";
ctx.fillRect(50, 50, 200, 100);

// Desenhando um retângulo apenas com as bordas
ctx.strokeStyle = "blue";
ctx.lineWidth = 2;
ctx.strokeRect(50, 50, 200, 100);

// Desenhando uma linha
ctx.beginPath();
ctx.moveTo(300, 50);
ctx.lineTo(400, 150);
ctx.stroke();

// Desenhando texto
ctx.font = "24px Arial";
ctx.fillStyle = "black";
ctx.fillText("Hello, Canvas!", 50, 200);

Métodos e Propriedades:

  • getContext(): retorna o contexto de renderização 2D ou 3D do canvas.
  • fillRect(x, y, width, height): preenche um retângulo com uma cor sólida.
  • strokeRect(x, y, width, height): desenha a borda de um retângulo.
  • clearRect(x, y, width, height): limpa uma área retangular do canvas.
  • beginPath(): inicia um novo caminho (path).
  • moveTo(x, y): move o ponto atual do caminho para as coordenadas especificadas.
  • lineTo(x, y): adiciona uma linha reta do ponto atual até as coordenadas especificadas.
  • arc(x, y, radius, startAngle, endAngle, anticlockwise): desenha um arco.
  • fill(): preenche o caminho atual com a cor de preenchimento atual.
  • stroke(): desenha a borda do caminho atual.
  • drawImage(image, x, y): desenha uma imagem no canvas.
  • save(): salva o estado atual do contexto de renderização.
  • restore(): restaura o estado do contexto de renderização anteriormente salvo.
  • translate(x, y): move a origem do sistema de coordenadas para um novo ponto.
  • rotate(angle): rotaciona o sistema de coordenadas em torno da origem.
  • scale(scaleX, scaleY): escala o sistema de coordenadas em relação à origem.
  • canvas.width: define a largura do canvas.
  • canvas.height: define a altura do canvas.
  • canvas.style: define o estilo CSS do canvas.
  • context.fillStyle: define a cor ou padrão de preenchimento do contexto.
  • context.strokeStyle: define a cor ou padrão da borda do contexto.
  • context.lineWidth: define a largura da linha do contexto.
  • context.font: define a fonte do texto a ser desenhado.
  • context.globalAlpha: define a transparência global do contexto.
  • context.globalCompositeOperation: define como as formas são compostas umas sobre as outras.
  • context.shadowBlur: define o desfoque da sombra do contexto.
  • context.shadowColor: define a cor da sombra do contexto.
  • context.shadowOffsetX: define o deslocamento horizontal da sombra do contexto.
  • context.shadowOffsetY: define o deslocamento vertical da sombra do contexto.

API Fullscreen

A API Fullscreen permite exibir elementos HTML em modo de tela cheia. Ela fornece métodos e eventos para entrar e sair do modo de tela cheia.

Exemplo de código:

// Verifica se o modo de tela cheia está disponível
if (document.fullscreenEnabled) {
  // Obtém um elemento do DOM para entrar em modo de tela cheia
  const element = document.getElementById("myElement");

  // Solicita entrar em modo de tela cheia quando um evento ocorrer
  element.addEventListener("click", function() {
    if (element.requestFullscreen) {
      element.requestFullscreen();
    } else if (element.mozRequestFullScreen) { // Firefox
      element.mozRequestFullScreen();
    } else if (element.webkitRequestFullscreen) { // Chrome, Safari e Opera
      element.webkitRequestFullscreen();
    } else if (element.msRequestFullscreen) { // Internet Explorer e Edge
      element.msRequestFullscreen();
    }
  });

  // Sai do modo de tela cheia quando um evento ocorrer
  document.addEventListener("fullscreenchange", function() {
    if (!document.fullscreenElement) {

      console.log("Saiu do modo de tela cheia");
      // Lógica adicional após sair do modo de tela cheia
    }
  });

  // Manipulando erros ao entrar no modo de tela cheia
  document.addEventListener("fullscreenerror", function(event) {
    console.error("Erro ao entrar no modo de tela cheia", event);

  });
